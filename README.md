# Plan Tracker

A personal planning and progress tracking tool — no install, no backend, no dependencies.

Open `index.html` in any browser — no install needed.

## Features

- Create unlimited plans with phases and steps
- Check off steps as you complete them
- Progress bars per phase and overall
- Dark mode (follows system preference)
- All data saved locally in your browser (localStorage)
- Preloaded with a Nonprofit SaaS Roadmap plan

## Deploy to GitHub Pages

1. Push `index.html` and `README.md` to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch and **/ (root)** folder
5. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>/`

That's it. No build step, no CI config needed.
